# 图书资源分享（静态站点）

这是一个简单的静态图书资源分享网站示例，适合部署到 GitHub Pages。

包含文件：

-   `index.html`：首页，展示书籍列表并支持搜索/筛选
-   `book.html`：书籍详情页面
-   `books.json`：书籍数据（示例）
-   `assets/css/styles.css`：样式表
-   `assets/js/main.js`：客户端渲染脚本

部署到 GitHub Pages：

1. 将本目录的内容推送到 GitHub 仓库的 `main` 分支（或 `gh-pages` 分支）。
2. 在仓库设置中启用 GitHub Pages，选择 `main` 分支的根目录作为发布源。
3. 等待几分钟，访问 https://<your-username>.github.io/<repo-name>/ 即可。

说明：示例书籍使用了占位图与示例下载链接，请替换为真实资源与封面图。
